SWE-Fixer: Training LLMs for Real World Github Issue Resolving

TLDR: We introduce SWE-Fixer, an effective solution for resolving real-world GitHub issues. It demonstrates competitive performance on the SWE-Bench Lite benchmark. We plan to make our code, model, and data publicly available.
